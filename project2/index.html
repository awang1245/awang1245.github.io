About

This site is designed to be a fun, audiovisual guide to help users identify some common birds in the Providence area. I hope that this guide can help you learn more about the feathered friends you may hear in your daily life.

Designed and coded by Anna Wang for Web as Medium 1 @ RISD, advised by Minkyoung Kim.

Feedback

Bugs to catch? Questions, comments, or suggestions? Fill out this form.

I'm hoping to add seasons and more birds in the future, along with increased accessibility. This site still has many ways to grow, so any and all feedback is appreciated.
